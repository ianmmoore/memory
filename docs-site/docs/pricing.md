## FAQ

**Can I change plans?**
Yes, upgrade or downgrade anytime. Changes take effect immediately.

**What happens if I exceed my limits?**
On Free: Requests are blocked. On paid plans: Overage charges apply.

**Is there a free trial?**
Starter and Professional include a 14-day free trial.

**How is usage calculated?**
- API calls: Each HTTP request
- Tokens: Input + output tokens for LLM operations
- Memories: Peak storage during billing period
